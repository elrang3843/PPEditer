# PPEditer 변경 히스토리

---

## v1.9 — 2026-04-22

### 슬라이드 패널 클립보드
- **슬라이드 복사** (`Ctrl+C`) — 슬라이드 패널에서 선택한 슬라이드를 클립보드에 복사
- **슬라이드 잘라내기** (`Ctrl+X`) — 선택한 슬라이드를 클립보드에 복사 후 삭제 (마지막 슬라이드는 잘라내기 불가)
- **슬라이드 붙여넣기** (`Ctrl+V`) — 현재 선택한 슬라이드 바로 뒤에 복사본 삽입; 이미지 포함 슬라이드 완전 복원
- **슬라이드 삭제** (`Delete`) — 확인 후 삭제 (기존 메뉴와 동일)
- 우클릭 컨텍스트 메뉴에 잘라내기 / 복사 / 붙여넣기 / 삭제 항목 추가; 붙여넣기·삭제는 상황에 따라 비활성화
- 우클릭 시 자동으로 해당 슬라이드 선택

## v1.8 — 2026-04-22

### 탬플릿 기능
- **탬플릿 불러오기** — 기존 `.pptx` 파일을 탬플릿 편집 모드로 열기 (탬플릿 메뉴)
- **탬플릿으로 새로 만들기** — 탬플릿 목록에서 선택해 새 프레젠테이션 생성 (내용은 복사, 파일 경로 없음)
- **탬플릿 편집 모드** — 제목 표시줄에 `[탬플릿 편집 중]` 표시, 모드 전환 가능
- **탬플릿 저장 / 다른 이름으로 저장** — 표준 탬플릿 폴더(`%AppData%\PPEditer\Templates\`)에 저장
- **탬플릿 편집 모드 종료** — 일반 편집 모드로 복귀
- **탬플릿 선택 대화상자** — 표준 탬플릿 폴더의 `.pptx` 파일을 목록으로 표시, 직접 찾아보기 버튼 제공
- 나머지 모든 편집 요소(도형, 텍스트, 슬라이드, 효과 등) 그대로 사용 가능

## v1.7 — 2026-04-22

### 편집 기능
- **개체 복사 / 잘라내기 / 붙여넣기** (`Ctrl+C / Ctrl+X / Ctrl+V`) — 여러 개체 선택 시 일괄 처리
- **여러 개체 동시 삭제** — `Delete` 키로 다중 선택된 모든 개체를 한 번에 삭제

### 버그 수정
- **텍스트 편집 커서가 위치에 따라 보이지 않는 문제 수정**
  - 편집기(`RichTextBox`)를 `Viewbox` 바깥 독립 오버레이(`EditorOverlay`)로 이동
  - `TranslatePoint`로 화면 픽셀 좌표를 직접 계산하여 캐럿이 항상 정수 픽셀에 배치됨
- **편집 중 글자 크기가 보기 모드와 달라 보이는 문제 수정**
  - `FlowDocument` 폰트 크기를 `Viewbox` 스케일 비율로 조정, 저장 시 역변환하여 PPTX 데이터 보존
- **편집 모드 해제 직후 크기 조절이 안 되는 문제 수정**
  - `CommitEdit` 호출 중 캔버스가 재생성될 때 마우스 핸들러가 구 캔버스를 계속 참조하던 버그 수정
- **수식 / 특수문자 / 이모지 메뉴 항목이 편집 모드 반복 후 영구 비활성화되는 문제 수정**
  - 편집 시작 시 `UpdateActions()` 추가 호출
  - Insert 메뉴 열림 시 `LostFocus`가 편집을 조기 종료하던 문제 해결 (`MenuBase` 포커스 이동 예외 처리)

## v1.6 — 2026-04-22

### 도형 및 편집
- **도형 회전** 추가 (우클릭 컨텍스트 메뉴 → 회전)
- **방향 인식 그리기**: 평행사변형·직각삼각형·화살표·호가 드래그 방향에 따라 자동으로 올바른 방향으로 그려짐
- **텍스트 상자 편집 진입 시 자리 표시자 텍스트 자동 전체 선택**

### 프레젠테이션
- **슬라이드 노트** 기능 추가 (`보기 > 노트 표시/숨김`): 편집·발표자 화면·인쇄 모두 지원
- **발표자 도구(Presenter View)** 추가: 발표자 화면에서 현재/다음 슬라이드·노트·경과 시간 동시 확인
- **모니터 설정** 추가 (`설정 > 모니터 설정`): 발표 화면과 발표자 화면에 표시할 모니터 개별 선택

### 워터마크
- **워터마크 기능** 추가 (`보기 > 워터마크`): 텍스트 워터마크를 편집 화면·슬라이드 쇼·인쇄 출력에 표시
- 슬라이드 쇼 및 인쇄 시 `RenderTargetBitmap` 방식으로 안정적 렌더링

### 문서 정보 개선
- 기존 파일 열기 시 제목·주제·작성자 등이 비어 있던 문제 수정 (`PackageProperties` API 전환)
- **수정자(LastModifiedBy) 편집 가능**: 문서 정보 대화상자에서 작성자와 수정자 모두 직접 편집
- **저장 시 기본값**: 작성자/수정자가 비어 있을 때만 Windows 계정 이름으로 자동 설정

### 색상 선택기 전면 개선
- SV 평면(채도·명도) + 색조(Hue) 세로 스트립 + R/G/B 슬라이더 + HEX 입력 + 미리 보기로 구성된 전문적인 HSV 색상 선택기

### 자동 저장
- **자동 저장** 기능 추가 (`설정 > 자동 저장`): 1 / 2 / 5 / 10 / 30분 간격 선택
- 저장 경로가 없는 파일의 경우 저장 위치 지정 알림 옵션

### 도움말
- **라이선스 정보** 대화상자 추가 (`도움말 > 라이선스 정보`)

### 버그 수정
- 폰트 필터: 상용 여부 감지 방식을 `fsType` 비트 필드 기반으로 개선
- PDF 내보내기 `MemoryStream` 직접 접근 오류 수정
- 호(Arc) 끝점 좌표 계산 오류 수정
- 스프라인 도형 클리핑 문제 수정 (컨테이너를 `Grid` → `Canvas`로 교체)
- 선택 오버레이가 회전된 도형을 따르지 않던 문제 수정

## v1.4 — 2026-04-21

### 슬라이드 쇼 (F5)
- **전체 화면 슬라이드 쇼** 추가 (`슬라이드 > 슬라이드 쇼` 또는 `F5`)
- 탐색: `PageDown / Space / → / ↓ / 마우스 왼쪽 클릭` → 다음, `PageUp / Backspace / ← / ↑ / 마우스 오른쪽 클릭` → 이전, `Esc` → 종료

### 슬라이드 전환 효과
- 6종 전환 효과: 밝기변화(Fade) / 밀어내기(Push) / 나타나기(Wipe) / 집어내기(Flip) / 구겨던지기(Crumple) / 모핑(Morph)
- **시간 설정** (0.1초 단위), 모든 슬라이드 일괄 적용 옵션
- PPTX 표준 `<p:transition>` 요소로 저장 (Morph 제외)

### 개체 애니메이션 효과
- 5종 애니메이션: 밝기변화(FadeIn) / 날아오기(FlyIn) / 펄스(Pulse) / 튕김(Bounce) / 닦아내기(WipeIn)
- **시간 설정** (0.1초 단위), **반복 횟수** (0=무한 반복), **자동 실행** 옵션 (슬라이드 표시 시 즉시 재생)
- 개체 애니메이션 메뉴 — 도형 선택 시 활성화되도록 선택 이벤트 연동 수정

### 버그 수정
- 슬라이드 전환·애니메이션 효과가 동작하지 않던 문제 수정 (`Storyboard` → `BeginAnimation` 직접 호출 방식으로 전면 교체)
- 전환/애니메이션 다이얼로그 하단 버튼이 가려지는 레이아웃 문제 수정 (`DockPanel` 구조로 개선)

- Python 프로토타입에서 **C# + WPF (.NET 8)** 으로 전면 재구현 (라이선스 문제 해결)

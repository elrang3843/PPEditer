## v1.4 — 2026-04-21

### 슬라이드 쇼 (F5)
- **전체 화면 슬라이드 쇼** 추가 (`슬라이드 > 슬라이드 쇼` 또는 `F5`)
- 탐색: `PageDown / Space / → / ↓ / 마우스 왼쪽 클릭` → 다음, `PageUp / Backspace / ← / ↑ / 마우스 오른쪽 클릭` → 이전, `Esc` → 종료

### 슬라이드 전환 효과
- 6종 전환 효과: 밝기변화(Fade) / 밀어내기(Push) / 나타나기(Wipe) / 집어내기(Flip) / 구겨던지기(Crumple) / 모핑(Morph)
- **시간 설정** (0.1초 단위), 모든 슬라이드 일괄 적용 옵션
- PPTX 표준 `<p:transition>` 요소로 저장 (Morph 제외)

### 개체 애니메이션 효과
- 5종 애니메이션: 밝기변화(FadeIn) / 날아오기(FlyIn) / 펄스(Pulse) / 튕김(Bounce) / 닦아내기(WipeIn)
- **시간 설정** (0.1초 단위), **반복 횟수** (0=무한 반복), **자동 실행** 옵션 (슬라이드 표시 시 즉시 재생)
- 개체 애니메이션 메뉴 — 도형 선택 시 활성화되도록 선택 이벤트 연동 수정

### 버그 수정
- 슬라이드 전환·애니메이션 효과가 동작하지 않던 문제 수정 (`Storyboard` → `BeginAnimation` 직접 호출 방식으로 전면 교체)
- 전환/애니메이션 다이얼로그 하단 버튼이 가려지는 레이아웃 문제 수정 (`DockPanel` 구조로 개선)

## v1.3 — 2026-04-20

### 도형 렌더링 개선
- **사용자 정의 도형 경로(custGeom) 렌더링** 수정 — Line·PolyLine 등이 사각형으로 잘못 표시되던 문제 해결
  - `a:custGeom` 경로 데이터(MoveTo, LineTo, CubicBezierCurveTo) 정확히 파싱·렌더링
  - 타원형 프리셋(`ellipse`) → WPF `Ellipse`, 그 외 → `Rectangle`로 구분 처리

### 버그 수정
- 색상 선택기 팔레트 클릭 시 즉시 선택 및 닫힘 처리
- 도형 속성 변경(채우기/테두리) 후 화면 미반영 문제 수정
- 텍스트 편집 진입 시 기존 서식이 초기화되던 문제 수정
- 편집 완료 후 텍스트 내용이 복원되지 않는 문제 수정
- `Paragraph.Margin` 음수 EMU 값으로 인한 `ArgumentException` 방어 처리
- 편집 모드 진입 시 원본 도형과 편집기가 겹치는 문제 수정

## v1.1 — 2026-04-15

### 텍스트 서식 편집
- **글자 속성** 다이얼로그 (`Ctrl+Shift+F`): 글꼴, 크기, 굵게/기울임/밑줄, 색상, 위·아래 첨자
- **문단 속성** 다이얼로그 (`Ctrl+Shift+P`): 정렬(왼/가운데/오른/양쪽), 줄 간격, 단락 앞/뒤 간격, 들여쓰기
- **문서 정보** 다이얼로그 (`Ctrl+Shift+I`): 슬라이드 크기, 방향, 열기 암호 설정
- **텍스트 상자** 그리기 추가 (`Ctrl+Shift+T`)
- `PptxConverter` 양방향 변환 개선: 단락 기본 런 속성 상속, 명시적 값만 기록

### 버그 수정
- 텍스트 정렬 관련 `TextAlignmentTypeValues` SDK v3 구조체 대응
- `ModifyVerifier` SDK v3 접근 불가 → `OpenXmlUnknownElement` + `SetAttribute` 방식으로 우회
- 여러 빌드 오류 수정 (네임스페이스 충돌, 델리게이트 서명, 열거형 이름 등)

## v1.0 — 2026-04-10

### 초기 구현
- Python 프로토타입에서 **C# + WPF (.NET 8)** 으로 전면 재구현 (라이선스 문제 해결)
- **PPTX 파일** 열기, 편집, 저장
- **슬라이드 패널** (왼쪽 썸네일 목록)
- **인라인 텍스트 편집** (더블클릭)
- **슬라이드 추가 / 삭제 / 복제 / 순서 변경**
- **실행 취소 / 다시 실행** (최대 50단계, 전체 문서 직렬화 방식)
- **PDF 내보내기** (PDFsharp, 150 DPI)
- **확대/축소** (25%~300%, 창 맞춤)
- **최근 파일** 목록 (레지스트리, 최대 10개)
- **OpenXML SDK v3.2** 대응 (구조체 타입, `EnumValue<T>` 등 호환 처리)

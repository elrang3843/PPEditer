# CHANGELOG

PPEditer 버전별 변경 이력입니다.

---

## [v0.3] — 2026-04-21

### 추가
- **그룹/언그룹** (편집 메뉴, `Ctrl+G` / `Ctrl+Shift+G`)
  - Ctrl+클릭으로 다중 선택 후 묶기
  - GroupShape 렌더링 지원
- **개체 속성 대화상자 개편** — 3탭 구성
  - 위치/크기 탭
  - 채우기 탭 (종류 + 색상 선택)
  - 테두리 탭 (종류 / 두께 / 색상)
- **색상 선택 팝업** (`ColorPickerPopup`) — 24색 스와치 + 16진수 입력
- **문자 속성 대화상자** (편집 메뉴, `Ctrl+Shift+F`) — 2탭 구성
  - 글꼴 탭: 폰트·크기·간격·전경색·배경색·줄 색
  - 스타일 탭: 두껍게·기울이기·외곽선·밑줄·삭제선·윗줄·위/아래첨자
- `PptxConverter` 문자 속성 왕복 변환 (취소선·첨자·간격·강조색·줄 색·외곽선)

### 수정
- 툴바 버튼 `DynamicResource` 표기 오류 — 태그 내 텍스트 → `Content=` 속성으로 수정
- `A.OpenXmlElement` 네임스페이스 오류 수정
- SDK v3 `ShapeTypeValues.IsoscelesTriangle` → `new A.ShapeTypeValues("isoTri")`
- `PresetLineDashValues.DashDotDot` → `LargeDashDotDot` (DrawingML 미지원 값 수정)
- `OnGroup` / `OnUngroup` XAML `RoutedEventHandler` 시그니처 불일치 수정
- `TextElement` / `RgbColor` 네임스페이스 충돌 (`using` 별칭으로 해결)

## [v0.2] — 2026-04-21

### 추가
- **도형 그리기** — 18가지 도형 지원
  - 직사각형·정사각형·타원·원·삼각형류(등변·이등변·직각·부등변)
  - 사다리꼴·평행사변형·호·화살표·다각형·스플라인 폴리곤
  - 직선·꺾은선·스플라인선
  - 도형 그리기 중 실시간 미리보기
- **Z-순서 컨텍스트 메뉴** — 맨 앞으로·앞으로·뒤로·맨 뒤로 4단계
- **수식 기호 삽입 대화상자** (`Ctrl+Shift+M`)
- **개체 속성** 팝업 (위치/크기 편집)
- **오른쪽 클릭 메뉴** — 개체 속성, Z-순서 이동
- **문자 지도** (`Ctrl+Shift+K`) — 6개 카테고리 특수 문자 삽입
- **이모지 선택기** (`Ctrl+Shift+J`)
- **화살표 키 도형 이동** (1 EMU 단위)
- **문서 정보 편집기** (편집 → 문서 정보, `Ctrl+Shift+I`)

### 수정
- 이미지 드래그 이동 버그 수정
- 시작 시 `FontCombo` NullReferenceException 수정
- SDK v3 빌드 오류 다수 수정
